class TopicModel.TopicModel(name, preprocessing_object, num_topics=20)¶

Class for Topic models including training, getting word-topic distribution and document-topic distribution. Model implemented : LDA, LDA mallet (and CTM if time permit it)

LDA_topic_visualization(filename)¶

Function inspired by : https://www.machinelearningplus.com/nlp/topic-modeling-visualization-how-to-present-results-lda-models/ Plot the TSNE-clustering to represent the 2D projection of each document and clustering them based on the topics where the probability is the highest inside the document-topic distribution

Parameters

filename – path to save the figure
