LDA analysis¶

LdaAnalysis.analysis(model, iteration, corpus='BoW')¶

Train different topic model with a specific number of iteration and with different number of topics

Parameters

  • model – TopicModel object

  • iteration – iteration wanted for the analysis

  • corpus – corpus served to train the model

Returns

list of all trained models

LdaAnalysis.grid_search(topicModel, topic_list, iteration)¶

Calculate and return evaluation score for model trained with different number of topics

Parameters

  • topicModel – topicModel object

  • topic_list – list of all topics wanted for the grid search

  • iteration – number of iterations wanted for all trained model

Returns

grid: a dictionary with all following content

coherence_CV: list of CV coherence for each model trained during the grid search coherence_UMass: list of UMass coherence for each model trained during the grid search model: list of each model trained during the grid search topic_similarity: each model trained during the grid search cosine_similarity: list of the cosine similarity between topics (see the function dist_cos) on each model trained during the grid search

LdaAnalysis.topic_cosine_similarity(topicModel)¶

Average of all cosine distance of all word-topics distribution

Parameters

topicModel – topicModel object

Returns

average cosine distance between two random topics from a topic model
