class Preprocess.Preprocessing1(*arg)¶

Preprocessing 1 based on Synset and Lemmatization

execute()¶

Doing the preprocessing 1 on the text. Tokenizes the document and a list of bigram is made for them. Then finds the first synset for each word/tag combination. If a synset is not found for that combination it is skipped.

execute()¶
